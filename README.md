## Overview
LEAFORA is a marketplace-style web platform that enables users to buy, sell, and rent books with secure authentication, order tracking, and real-time notifications.

## Problem Statement
Students and readers often struggle to:
- Find affordable second-hand books
- Sell unused books easily
- Track rental and purchase orders in one place

LEAFORA solves this by providing a centralized, user-friendly book exchange platform.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Database: SQLite
- Template Engine: Jinja2
- Authentication: Session-based login

## Key Features
- User authentication & session management
- Buy / Sell / Rent books
- Order management & history
- Real-time notifications
- Personalized dashboard

## Architecture
- MVC-inspired Flask structure
- Modular route handling
- Secure session-based authentication
- Relational database design using SQLite
